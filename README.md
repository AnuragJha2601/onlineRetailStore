# ✨ Dhanak Trinket - Ethnic Finds, Timeless Shine

Welcome to **Dhanak Trinket**, a modern e-commerce platform specializing in exquisite ethnic finds including bangles, necklaces, imitation jewelry, and traditional accessories. Our curated collection features handpicked designs that blend traditional craftsmanship with contemporary style, with plans to expand into ethnic wear and cultural artifacts.

## 🌟 Features

### Current Features
- **📱 Responsive Design**: Seamless shopping experience across all devices
- **🖼️ High-Quality Images**: Crystal-clear product photography with zoom functionality  
- **❤️ Product Likes**: Show appreciation for your favorite pieces
- **📦 Stock Status**: Real-time inventory updates with "Sold Out" indicators
- **🔍 Easy Navigation**: Intuitive category browsing (Bangles, Necklaces, Earrings, etc.)
- **⚡ Fast Loading**: Optimized performance for quick browsing

### Coming Soon

## 🔒 Security & Deployment

### Security First Approach
- **🛡️ No Secrets in Code**: All sensitive data configured securely in Azure
- **🔐 Azure AD Authentication**: Database connections use managed identity
- **🌐 Secure CORS**: Restricted to verified domains only
- **📊 Monitoring**: Application Insights tracks security events
- **🔒 HTTPS Only**: All traffic encrypted end-to-end
